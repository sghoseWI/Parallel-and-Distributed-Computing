    In this assignment, you will execute on an individual project that accomplishes the following:

    • Generates insight into a reasonable question or problem that has clear implications for policy or social impact (if you're uncertain, please just run it by me);
    • Implements task parallelism in Python, that:
      • Ingests a stream of text data (though the stream can be somewhat contrived).
      • Uses at least three mp.Process objects working in (at least) two stages of tasks (so a minimum of two producers and one consumer, or one producer and two consumers);
      • Uses mp.Queue to pass data between the producer and consumer processes;
      • Deploys CoreNLP to an AWS EC2 instance (more guidance will be attached to this post soon), and uses that server to run natural language processing functions on your data stream. Note: most of your code can run on your local computer, while only sending requests to the deployed EC2 instance (meaning, you do not have to run all this code on the EC2 instance).
      The final submission should be a (1) Python Notebook (please submit the notebook in .ipynb format and an HTML export) containing the code and results of running the analysis, along with clear and explicit writing detailing the process and analyzing the outcomes. You should also submit (2) the AWS CLI command you are using to launch your EC2 instance.

      There is a thorough introduction to installations in AWS EC2 on my GitHub, as well as specific instructions for installing CoreNLP on Canvas (though you need to automate the process with the AWS CLI): CoreNLP-SetUp.pdf

      An Example Project 

      Below, I have a picture of one complete example project that would fulfill the obligations of this assignment. You can see two processes scraping articles from a news feed (e.g. RSS feeds) and then potentially doing some additional data cleaning. These 'producer' processes then pass some data to queue (mp.Queue), which can then be picked up from (.got) a consumer process. The consumer process may do multiple things, but it must use the CoreNLP server that you have setup on an EC2 instance (sending Post requests will work and return results to your local computer). 

      What Can Natural Language Processing Do?

      For those who have not done much of it, CoreNLP’s Natural Language Processing can perform tasks like:

      • Estimate sentiment (e.g. how positive or negative is a tweet, see this blog post on Trump’s tweets);
      • Recognize proper nouns (called Named Entity Recognition, e.g. useful in identifying focus of political speeches – are they mentioning Iran? Russia? ISIS?);
      • Analyze word frequencies (e.g. you might learn that papers written by social scientists use the phrase “independent variables” while data scientists say “features”).
      • Use words as features in a predictive model (e.g. using Yelp reviews to predict health code violations, which recently expanded nationwide and is quite controversial). Note you need some labeled data to train your model;
      • Identify common clusters of words within documents (called topic modeling, e.g. in running topic models on policy documents, some might all contain “health”, “doctor”, “patient”, “hospital”, suggesting they are about healthcare.
      • Additionally, neural network driven methods of NLP can go much further, but are beyond the scope of this assignment.

      You can read more in the CoreNLP documentation.

Deploys and Queries CoreNLP 2/3 Given how little data there is here, it might have been more interesting to analyze the syntactic structure of the text. Given your question, I just don't think sentiment makes a lot of sense. Final Grade 7/10 This has all the required components of the assignment, but frankly needed more time and attention to be a strong proof-of-concept.
